### Validity criteria (same as `analysis_pipe/src/validity_gates.R`)

- **Valid** if all hold:
  - image tracking rate >= `0.70`
  - word sensitivity rate >= `0.20`
  - parse quality >= `0.97`
- **Borderline** if image tracking rate >= `0.50` but not fully valid
- **Invalid** otherwise

**How R assigns the label (`case_when` order in `validity_gates.R`):** a row is **`valid`** only if all three valid thresholds pass. If not, it is **`borderline`** when `image_tracking_rate >= 0.50`, and **`invalid`** when `image_tracking_rate < 0.50`. So every **`invalid`** model necessarily fails the **0.50** tracking floor first; other metrics can fail at the same time but do not change that tier.
